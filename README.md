💻 Calculadora de Gorjeta

Projeto desenvolvido para a disciplina Programação para Internet (ILP951) da FATEC Jahu.

 Descrição

Esta aplicação web permite calcular automaticamente o valor da gorjeta com base em:

Valor total da conta
Quantidade de pessoas
Percentual de gorjeta

Além disso, o sistema classifica o pagador como:

 Mão de vaca (menos de 5%)
 Legal (entre 5% e 15%)
 Generoso (acima de 15%)

 Como executar o projeto
Clone o repositório:
git clone https://github.com/seu-usuario/seu-repositorio.git
Acesse a pasta do projeto:
cd calculadora-gorjeta
Instale as dependências:
pip install flask
Execute a aplicação:
python app.py
Acesse no navegador:
http://127.0.0.1:5000
